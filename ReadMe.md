# Foobar

Canvas plotter is an interactive plotting system based on data

## Installation

To install this project, installing it locally using npm:
```bash
cd root
npm install
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
